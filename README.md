# PumpScan

PumpScan is a modern, responsive React Native application that allows users to scan, track, and monitor various crypto tokens to evaluate their safety and market performance.

## Features
- **Token Search & Analysis**: Real-time token searching and safety scoring.
- **Trending Tokens**: View top tokens and their market cap natively on the home screen.
- **Watchlist**: Track your favorite tokens securely using your account.
- **Alerts**: Stay informed with custom notifications for market changes.

## Development Setup

1. **Clone the repo**
   ```bash
   git clone https://github.com/Muhammad-Hassan-X/PumpScan.git
   ```
2. **Install dependencies**
   ```bash
   cd client
   npm install
   ```
3. **Start the app**
   ```bash
   npm start
   ```

## Tech Stack
- Frontend: Expo, React Native, NativeWind
- Backend: Node.js, Express, Supabase (PostgreSQL)
